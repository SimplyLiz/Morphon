MORPHON ENGINE

A living neural network that grows, learns, and self-organizes at runtime inside a 3D Poincaré ball. Center = general, surface = specialized.

Cell Types

Stem — pluripotent, undifferentiated
Sensory — input processing
Associative — pattern recognition
Motor — output generation
Modulatory — internal regulation
Fused — cluster member

Signals

  • Reward — reinforces recently active connections (dopamine)
  • Novelty — increases plasticity systemwide (acetylcholine)
  • Arousal — increases sensitivity (noradrenaline)

Keyboard Shortcuts

  • Space Pause / Resume
  • → Single step
  • R Reset system
  • F Fullscreen
  • Esc Deselect / Close help
  • ? Toggle this help

Interaction

  • Orbit — drag to rotate, scroll to zoom
  • Hover — tooltip with morphon info
  • Click node — select, highlight connections
  • Click cell type — highlight all of that type
  • Click empty — deselect
