\section{Experiments}
\label{sec:experiments}

We evaluate MI on two tasks chosen to probe different capabilities: reactive control (CartPole) and static classification (MNIST). The goal is not to claim state-of-the-art accuracy on any single task but to demonstrate that a self-organizing developmental architecture can solve non-trivial problems and to characterize its distinctive behaviors.

\subsection{CartPole-v1}
\label{sec:exp_cartpole}

\paragraph{Setup.}
OpenAI Gym CartPole-v1, 4 observations (cart position, velocity, pole angle, angular velocity) encoded into 32 sensory morphons via Gaussian population coding (8 receptive-field tiles per observation). 2 motor morphons (left/right force). Cerebellar developmental preset: seed\_size=60, dimensions=4, initial\_connectivity=0.25.

A small linear TD-error critic external to the MI network provides the reward modulation signal, biologically modelling dopamine neurons in VTA/SNc that compute $\delta = R + \gamma V(s') - V(s)$ from striatal input \cite{fremaux2013reinforcement}. The critic is the only component using exact gradients; everything inside the MI network is trained by three-factor learning.

\paragraph{Result.}
\textbf{Solved} on both v0.5.0 and v4.1.0 standard profiles. v0.5.0: avg(100)=195.2 at episode 895, best=468. v4.1.0: avg(100)=195.5 at episode 936, best=468. The Pulse Kernel Lite and iSTDP changes introduced for MNIST did not affect CartPole learning dynamics. The solve confirms that the full MI pipeline --- developmental bootstrap, three-factor learning, tag-and-capture consolidation, contrastive reward, Endoquilibrium regulation --- is stable across major architectural changes.

\paragraph{Readout architecture.}
\label{sec:readout_engineering}
The CartPole solve required identifying and fixing three independent failures in the analog readout (Section~\ref{sec:output_pathways}), each of which individually prevented learning:

\begin{enumerate}
  \item \textbf{Centered sigmoid} ($\sigma(p) - 0.5$ instead of $\sigma(p)$). The resting potential is zero, so $\sigma(0) = 0.5$. With $\sim$80 associative morphons, resting sigmoids sum to $\sim$40, dwarfing the state-dependent signal ($\sim$0.005 per morphon). Centering removes this offset.
  \item \textbf{Learnable per-output bias} $b_j$. Absorbs mean output level and frees the weights to encode only state differences.
  \item \textbf{No L2 weight decay}. With $\lambda = 0.001$ and 200 steps per episode, decay erases $\sim$18\% of learned weight per episode. Weight clipping to $[-5, 5]$ replaces it.
\end{enumerate}

Each fix individually moved CartPole from $\sim$10 steps toward solving. The diagnostic that surfaces each failure is \emph{output discrimination} $d_\pm$: $d_+ = V_1 - V_0$ for a right-leaning pole probe, $d_-$ for left-leaning. When $d_+$ and $d_-$ have opposite signs and $|d| > 5$, the readout is discriminative. With any fix missing, $d_+$ and $d_-$ remain same-signed --- the signature of a collapsed policy. This metric surfaced the readout architecture failure faster than any other diagnostic and is reusable across spiking architectures.

\paragraph{What this proves.}
The full MI pipeline --- developmental bootstrap, three-factor learning, tag-and-capture consolidation, contrastive reward, Endoquilibrium regulation --- works end-to-end on a non-trivial control task. CartPole is the prerequisite gate for all other benchmarks: if the basic learning loop is broken, nothing else can work. The CartPole solve provides the foundation to investigate harder tasks.

\subsection{MNIST}
\label{sec:exp_mnist}

\paragraph{Setup.}
Standard MNIST, 784 input pixels (28$\times$28), 10 output classes. Cortical developmental preset: seed\_size=200, dimensions=6, initial\_connectivity=0.02 (Phase 4 hardcodes 30\% S$\to$A density). Three training epochs of 5000 samples each (standard profile) or 3000 (quick profile). Analog readout (Section~\ref{sec:implementation}). Endoquilibrium enabled with the corrected stage-detection thresholds described in Section~\ref{sec:failure_modes}. Inhibitory competition via LocalInhibition mode with Vogels 2011 iSTDP (Section~\ref{sec:istdp}); fast path via Pulse Kernel Lite (Section~\ref{sec:pulse_kernel}). Table~\ref{tab:mnist} results from v4.1.0--v4.3.0 (online accuracy), seed=42. V3-SL results from v4.8.0, seed=42.

\paragraph{Analog readout rationale.}
A key design choice is using an analog readout (a linear layer over morphon potentials) rather than reading classification decisions from spike counts at motor morphons. The motivation: the spike pipeline destroys discriminative information. Morphon potentials carry the network's learned representation, but conversion to spikes, propagation through synaptic delays, leaky integration, and multi-hop accumulation at motor morphons each introduce lossy compression. A simple synthetic 2-class probe (bag-of-characters encoding, 27-dim input) confirmed this empirically: a spike-based motor readout on the same trained network produces chance-level accuracy ($\sim$46\%) while a linear analog readout over the same morphon potentials reaches 99\%. The information that distinguishes classes is present in the substrate --- the analog readout proves it --- but the spike pipeline cannot deliver it to motor morphons intact. The analog readout is therefore not a shortcut; it is the correct output mechanism for a spiking architecture where the spike pathway serves feature formation, not classification reporting. Biology achieves the same separation: Purkinje cells in the cerebellum perform analog integration of climbing and parallel fiber inputs rather than thresholding to binary spikes for motor control \cite{ito2006cerebellar}.

\paragraph{Epoch-0 as unsupervised exploration.}
A non-obvious but critical training protocol: reward signals (contrastive reinforcement and \texttt{report\_episode\_end}) are \emph{withheld} entirely during the first epoch (epoch 0). Reward is gated on \texttt{epoch > 0}. The reason: binary reward in epoch 0 allows the network to ``game'' the signal before class-selective representations have formed. Without inhibitory competition having had time to diversify activity patterns, a network that happens to favor one class early on will receive consistent positive reinforcement for that bias --- the reward signal reinforces the mode collapse instead of correcting it. Epoch 0 therefore functions as a pure unsupervised STDP phase: inputs drive feature formation via Hebbian coincidence detection, iSTDP balances excitation and inhibition, and no classification signal is applied. By epoch 1, morphon representations are sufficiently diverse that contrastive reward can usefully guide differentiation between classes rather than entrench an artifact. This distinction --- between a reward signal that guides and one that locks in early noise --- is a practical instance of the credit assignment problem in developmental learning: the credit must be withheld until the substrate is ready to use it.

\paragraph{Tag-and-capture consolidation: a latent bug fixed in v4.3.0.}
The tag-and-capture mechanism (Section~\ref{sec:architecture}) is the primary path by which synaptic weight changes survive structural plasticity events. A synapse is tagged when a coincident pre- and post-synaptic event occurs; it is consolidated (flagged as pruning-protected) when a reward signal arrives while the tag exceeds a threshold. In all versions prior to v4.3.0, the \texttt{capture\_threshold} parameter was set to $10.0$, while tags are stored in $[0, 1]$ --- a mismatch that made consolidation permanently impossible. Every synapse that should have been consolidated was silently pruned, including those that had been weight-updated during the most active classification learning in epochs 2--3. The Endo Differentiating stage (aggressive synaptogenesis, $cg = 2.0$) then pruned exactly these learned synapses, explaining the characteristic accuracy drop observed between epochs 2 and 3 in pre-fix runs. v4.3.0 sets \texttt{capture\_threshold = 0.5}, restoring the intended behavior: synapses that carry reward-correlated activity are protected from structural pruning and remain available as the substrate for long-term learning.

\paragraph{Hub dominance and its cure.}
The v3.0.0 baseline (30\%) was limited by a specific failure mode: a small number of ``hub'' morphons with very high in-degree accumulated nearly all reward-correlated weight updates and began responding to every input class, collapsing the readout to a near-random discriminator. With rate-based inhibitory STDP --- computed once per 10-tick medium period using a 100-tick activity average --- the inhibitory interneurons responded too slowly to break this dominance; a single hub could accumulate 50--100 correlated spikes before any corrective signal arrived.

Replacing rate-based iSTDP with the spike-timing rule of \citet{vogels2011inhibitory} fixes this at the root. On each delivered inhibitory spike (pre-spike): $\Delta w = -\eta \cdot x_{\text{post}}$, where $x_{\text{post}}$ is the post-synaptic trace. On each excitatory post-spike: $\Delta w = -\eta \cdot (x_{\text{pre}} - \alpha)$, where $\alpha = 2\rho_0\tau$ sets the equilibrium firing rate. The result: a hub morphon that fires at $5\times$ the target rate immediately recruits stronger inhibition on the very spikes that evidence its over-activity, rather than waiting for the slow running average to catch up. The readout response after this fix: 1 labeled class (class 4 only) $\rightarrow$ 12 labeled classes across all 10 digits, and accuracy 8\% $\rightarrow$ 44.5\%.

\paragraph{The self-healing finding revisited.}
\label{sec:self_healing_discussion}

In v3.0.0 (rate-based iSTDP), post-damage recovery reached \textbf{48\%} --- 18pp above the 30\% intact baseline. The mechanism: damage randomly removed entrenched hub morphons, Endoquilibrium responded by re-entering the high-plasticity Differentiating stage, and fresh morphons grew into well-specialized representations the hubs had blocked. The damage-and-regrow cycle was a manual proxy for hub clearing.

In v4.1.0, spike-timing iSTDP prevents hub formation during normal training, raising the intact ceiling to \textbf{52\%} on the standard profile. Post-damage recovery (1500 samples) reaches 28.5\% --- well below the intact baseline. This is expected: with no entrenched hubs to clear, damage is pure loss. The self-healing effect relied entirely on the pathology it was fixing.

This reframes the finding. The v3.0.0 self-healing result was not a feature of the architecture; it was a diagnostic. The damage cycle revealed that hub morphons were the limiting factor and that the substrate was capable of 48\% under the right conditions. That diagnostic pointed directly to the fix: inhibitory plasticity operating at the right timescale. With the fix in place, the system reaches 52\% without damage. The +18pp self-healing gain has been absorbed into the intact baseline.

\paragraph{Implication.}
The intact ceiling has risen from 30\% to 52\% (standard profile) via a single architectural correction: inhibitory plasticity at biological timescales. No tuning was required. The 22pp gain is entirely attributable to the iSTDP timescale fix; everything else (network size, training data, epochs) is unchanged. In v4.3.0, with the epoch-0 unsupervised phase, the capture\_threshold fix, and softer iSTDP parameters ($\eta_i = 0.001$, $w_i^0 = -0.2$), the quick-profile ceiling rises to 49\% median (peak 54\%) --- within 3pp of the standard-profile result.

\subsection{Stateless Training: Resolving LSM Co-adaptation (v4.8.0)}
\label{sec:stateless_training}

\paragraph{The co-adaptation problem.}
Even after the iSTDP fix, a diagnostic question remained open: why does standard sequential evaluation produce $\sim$52\% while offline classifiers on frozen features score only 10.5\%? The gap is real but its cause was mischaracterised in v4.4.0 as an intrinsic property of the LSM regime.

The root cause (identified in v4.8.0): sequential training co-adapts the readout to \emph{inter-image temporal context} rather than per-image discriminative structure. During training, the network accumulates transient state across images (membrane potentials, working memory, resonance queue). The readout learns to exploit that accumulated state. When evaluated on fresh inputs or in shuffled order, the exploited context patterns no longer hold and accuracy collapses.

\paragraph{Diagnostic: sequential vs.\ shuffled evaluation.}

\begin{center}
\begin{tabular}{lccc}
\hline
Variant & Sequential & Shuffled & $\Delta$ \\
\hline
V3 (standard training) & 52.0\% & 46.7\% & +5.3pp \\
V3 (ablation run)      & 54.0\% & 45.7\% & +8.3pp \\
\hline
\end{tabular}
\end{center}

The consistent sequential advantage quantifies the co-adaptation artefact. A per-image classifier that achieves 54\% sequentially but only 45.7\% on the same test set in shuffled order is exploiting the order, not the images.

\paragraph{Fix: stateless training (V3-SL).}
Call \texttt{System::reset\_transient\_state()} before each training image. The network can no longer exploit inter-image context during training; the readout must learn per-image discriminative representations. The reset zeroes membrane potentials, working memory, the resonance queue, episode fire counts, and ANCS context. Eligibility traces and synaptic weights are not reset.

\paragraph{Results.}
\begin{center}
\begin{tabular}{lccc}
\hline
Variant & Online & Stateless & Wall time \\
\hline
V3 (sequential training)  & 52.0\% & 81.3\% & 182s \\
\textbf{V3-SL (stateless training)} & \textbf{42.0\%} & \textbf{87.7\%} & \textbf{111s} \\
\hline
\end{tabular}
\end{center}

Stateless training raises the stateless metric by 6.4pp and is 39\% faster (111s vs 182s), because propagating accumulated sequential context is eliminated. Online accuracy drops from 52\% to 42\%: the correct interpretation is that 52\% online was inflated by co-adaptation, not that stateless training degrades the classifier. The correct evaluation metric for a per-image task on a stateful substrate is stateless accuracy.

\paragraph{Ablation: isolating stateless training vs.\ novelty-gain fix.}
Two changes shipped simultaneously in v4.8.0: stateless training and the novelty-gain fix (Section~\ref{sec:novelty_gain_collapse}, which suppressed the ACh modulation channel under energy pressure). An ablation re-enabling novelty suppression while keeping stateless training produced 88.0\% stateless --- +1.0pp above the joint result, within seed variance. Stateless training is load-bearing; the novelty-gain fix is an independent stability contribution that does not affect MNIST stateless accuracy.

\paragraph{Evaluation methodology.}
This finding has an implication for evaluation practice. \emph{Online accuracy} (sequential evaluation of a stateful substrate) is not a valid primary metric for per-image classification tasks: it measures, in part, how well the system exploits the evaluation protocol's order, which is an artefact. \emph{Stateless accuracy} (state reset before each test image) is the correct metric. The 82--88\% stateless range places MORPHON competitive with biologically-plausible STDP baselines such as Diehl \& Cook \shortcite{diehl2015unsupervised} (95\% with rate-coded inputs and a post-hoc label assignment protocol) while using a substantially harder regime: online learning, online readout, no separate label-assignment phase.

\subsection{Representational Regime: LSM Characterization}
\label{sec:lsm_characterization}

\paragraph{Offline feature evaluation.}
To characterize the representational regime of the trained system, we extract morphon activation vectors from the final V3 system (seed=42, standard profile, 5 epochs) after each of 1000 test images and train scikit-learn classifiers on the resulting feature matrix (80/20 train/test split). Both logistic regression and a two-layer MLP (512$\to$256 hidden units) achieve \textbf{10.5\%} --- random chance for 10 classes. This is not a measurement artifact.

\paragraph{Sensory-only readout ablation.}
To determine whether the associative layer contributes to the online accuracy or is computationally inert, we zero all readout weights connecting associative morphons to the output and re-evaluate with only Sensory$\to$Readout connections active. Results (same trained network, seed=42):

\begin{center}
\begin{tabular}{lcc}
\hline
Readout configuration & Accuracy & $\Delta$ \\
\hline
Full (Sensory + Assoc) & 48.5\% & baseline \\
Sensory-only (ablated) & 13.0\% & $-35.5$pp \\
\hline
\end{tabular}
\end{center}

The associative layer contributes \textbf{35.5pp} of the 48.5\% online accuracy. Zeroing its readout weights leaves the system at 13\% --- barely above random chance and consistent with a direct pixel-to-label linear projection with no learned intermediate representation.

\paragraph{Interpretation: Liquid State Machine with co-adapted readout.}
The combination --- 35.5pp associative contribution online, 10.5\% offline on frozen features --- is the signature of a \emph{Liquid State Machine} (LSM) \cite{maass2002real,lukosevicius2009reservoir}. The associative morphons function as a reservoir: they perform genuine computation that is discriminative for classification, but only in conjunction with the readout weights that were co-trained alongside the reservoir dynamics. An offline classifier sees static activation snapshots decoupled from the dynamical context in which those features are meaningful, and recovers no signal. Offline evaluation of frozen morphon features is not a valid diagnostic for MORPHON-class architectures; the correct evaluation is online with the co-adapted readout.

The $\sim$50\% online ceiling (v4.4.0) was not a capacity limit. It was a co-adaptation artefact: the online metric was partly measuring sequential context exploitation. Stateless training (Section~\ref{sec:stateless_training}) removes the confound, and the stateless metric reaches 87.7\% --- confirming that the reservoir \emph{is} computing discriminative per-image features, but the standard evaluation protocol was obscuring them.

\paragraph{Reproducibility (v4.8.0).}
All results use a deterministic \texttt{StdRng} seeded from \texttt{SystemConfig.rng\_seed}. Repeated runs at seed=42 (standard profile) produce results within $\pm$1pp. The V3-SL result of 87.7\% (seed=42) was reproduced at 87.0\% on an independent ablation run (same seed, ng suppression re-enabled), confirming the result is robust to the novelty-gain fix.

\label{sec:temporal}

\textbf{Delayed Match-to-Sample (DMS)} tests working-memory retention across blank delay periods. The system presents a pattern (5 steps), waits $N$ blank steps, then sees a probe and must report ``match'' or ``no match''. Achieving 91.7\% at delay=5 --- exceeding the 70\% target --- demonstrates that the recurrent reservoir retains pattern identity across multi-step delays without any explicit memory buffer: the dynamics persist in the recurrent activity.

\textbf{Sequence Classification} classifies a 4-element sequence by its temporal shape (rising/falling/peak/valley); each element is ambiguous in isolation. Length-8 sequences (88.8\%) are easier than length-4 (75.4\%) because the discriminative signal accumulates across more steps. The length-4 result (75.4\%) misses the 85\% target but is well above the 25\% chance level, confirming that temporal integration is functioning.

\textbf{Next-Element Prediction} tests whether the reservoir discovers periodicity from the reward signal alone. The near-zero MAE (0.0003 and 0.0025 for periods 3 and 5) indicates the system essentially memorises the periodic lookup table, consistent with the reservoir developing an attractor for each periodic position.

Together, these benchmarks confirm that MORPHON operates as a functional recurrent reservoir beyond static classification: it retains patterns across delays, integrates temporal context for shape discrimination, and discovers repeating structure. The DMS and sequence benchmarks are canonical neuroscience working-memory tests \cite{compte2000synaptic,baddeley2003working}; achieving human-competitive accuracy (90\%+) on DMS demonstrates that the bio-inspired architecture generalises to temporal paradigms without architectural modification.
