## 1. v2.0.0 Regression: Tag Accumulation

v2.0.0 introduced three changes that regressed CartPole from avg=195.2 (SOLVED) to avg=162.0 (not solved):

| Change | v0.5.0 | v2.0.0 | Impact |
|--------|--------|--------|--------|
| Tag accumulation | instant (`tag = 1.0`) | gradual (`tag += strength * 0.05`) | 20x slower credit assignment |
| Episode-end consolidation | single gate (above-average episode) | double gate (above-average AND above consolidation_gate) | blocks early consolidation |
| Receptor gating | binary (has receptor or not) | continuous sensitivity + meta-learning | noisy early sensitivity drift |

The tag accumulation slowdown was the primary cause. With 0.05 rate, a synapse needs ~20 reinforcing DFA updates to reach capture threshold vs 1 in v0.5.0. In a 200-episode quick profile, tags never accumulated fast enough.

## 3. The Tagging-and-Capture Architecture Decision

Initial fix modulated tag accumulation rate by endo stage (3x in Stressed). This was biologically wrong per Frey & Morris (1997) synaptic tagging-and-capture model:

- **Tags are LOCAL.** Set at the synapse by activity strength (calcium, STDP coincidence). Speed depends on local event magnitude, not global signals.
- **Capture is GLOBAL.** Requires plasticity-related proteins (PRPs) synthesized at the cell body in response to neuromodulators (dopamine, norepinephrine).
- **Capture = tag meets PRP.** Tagged synapse + available PRPs = consolidation. Untagged synapse ignores PRPs. Tagged synapse without PRPs decays.

Elevating tag rate during Stressed state diluted selectivity: bad-episode synapses got tagged just as aggressively as good-episode ones, and episode-gated capture couldn't distinguish them.

**Fix:** Removed `endo_tag_factor`. Added `consolidation_gain` channel to Endoquilibrium `ChannelState`. Tags accumulate at fixed local rate (0.3). Consolidation is endo-gated:

- Episode-end `delta_level *= consolidation_gain`
- DFA capture `effective_threshold = capture_threshold / consolidation_gain`

## 4. Dynamic Endoquilibrium Stage Detection

### 4.1 The Problem: Always Stressed

All v2.0.0 and early v2.1.0 runs showed the system stuck in `Stressed` stage for the entire run. The detection condition `pe_trend > 0.001 && pe_history.len() >= 100` triggered permanently because CartPole PE always rises from novel input trajectories. The system was "stressed by novelty," not by failure.

Result: `consolidation_gain=2.50` and `plasticity_mult=1.50` permanently. Endoquilibrium was a constant multiplier, not a regulation engine.

### 4.2 The Fix: Reward-Based Relative Thresholds

Replaced PE-based stage detection with reward-based detection using the system's own history as baseline:

```
Proliferating:   reward_history < 20 entries (too early to judge)
Stressed:        reward_trend < -0.01 * reward_slow_ema (reward DROPPING)
Mature:          reward_cv < 0.15 AND reward_trend ~ 0 (stable, low variance)
Consolidating:   reward_fast > 0.95 * reward_slow AND trend ~ 0 (near own ceiling)
Differentiating: reward_trend > 0 (improving)
```

No hardcoded absolute thresholds. Works for CartPole (reward = episode steps, range 10-500) and MNIST (reward = accuracy, range 0.0-1.0).

### 4.3 Stage-Dependent Consolidation and Plasticity

| Stage | plasticity_mult | consolidation_gain | Rationale |
|-------|----------------|-------------------|-----------|
| Proliferating | 1.5 | 2.5 | Learn everything fast, nothing to protect |
| Differentiating | 1.2 | 2.0 | Refining, still capturing aggressively |
| Consolidating | 0.8 | 1.0 | Slow learning, normal capture |
| Mature | 0.5 | 0.5 | Protect what works |
| Stressed | 1.5 | 0.5 | Explore freely, but DON'T consolidate bad patterns |

Key insight: Stressed has HIGH plasticity (reopen learning) but LOW consolidation gain (don't lock in weights during a crisis). Prior implementation had both high, which is correct for Proliferating (nothing to protect) but wrong for Stressed (prior learning at risk).

### 4.4 Observed Stage Progression in CartPole

Standard profile run (SOLVED ep 970, avg=195.4):

| Phase | Episodes | Dominant Stage | cg | pm |
|-------|----------|---------------|----|----|
| Bootstrap | 1-20 | Differentiating | 2.0 | 1.9 |
| Early plateau | ~100 | Mature | 0.5 | 0.6 |
| Improvement | 200-600 | Diff/Consol oscillation | 0.8-1.2 | varies |
| Mastered | 600-970 | Mature | 0.5 | 0.6 |
| Brief crisis (~550) | 1 episode | Stressed | 0.5 | 1.9 |

The system naturally decelerates as reward stabilizes. Consolidation_gain drops from 2.0 during active learning to 0.5 at mastery, protecting learned representations.

All CartPole runs use ~97 morphons (32 sensory, ~49 associative, ~14 modulatory, 2 motor). No structural plasticity occurs (no division, fusion, or clustering). The task is solved entirely through synaptic weight learning with the bootstrapped topology.

### 5.2 MNIST

V2 supervised readout achieves ~31% accuracy (3 epochs, 1000 images). This reflects the associative layer's feature quality ceiling, not the readout architecture. The morphon hidden layer produces near-identical activity patterns across input classes (same-activity problem diagnosed via firing rate diagnostics).

Non-accuracy metrics demonstrate V2 structural value:
- 32% lower prediction error vs V1
- 3x neuron coverage (fraction of associative morphons that fire)
- Full accuracy restoration after 30% associative damage (self-healing)

### 5.3 Architecture Validated

1. **Cerebellar readout pattern:** Unsupervised morphogenetic features + supervised climbing-fiber readout. Works for both RL (CartPole) and classification (MNIST).
2. **Tag-and-capture credit assignment:** Local tagging (fixed rate) + global capture (endo-gated). No backpropagation. Frey & Morris (1997) model implemented in silicon.
3. **Endoquilibrium regulation:** Reward-based developmental stage detection with relative thresholds. Genuine dynamic regulation replacing constant multipliers. System naturally accelerates during learning and decelerates at mastery.
4. **Consolidation_gain channel:** Novel endo channel controlling PRP availability. Stage-appropriate: aggressive during Proliferating/Differentiating, protective during Mature, cautious during Stressed.

## 6. Known Limitations

1. **CartPole solves slightly slower with dynamic endo** (ep 970 vs ep 918 with constant cg=2.5). This is correct behavior — the system decelerates at mastery — but means the quick profile (200 ep) won't solve.
2. **Stage oscillation during improvement phase.** The system flickers between Differentiating and Consolidating as reward crosses relative thresholds. The EMA smoothing prevents channel oscillation but the stage itself switches. Not harmful but noisy in logs.
3. **MNIST accuracy ceiling.** 31% with current associative layer features. Requires local inhibitory competition (replacing global k-WTA) to produce class-discriminative representations.
4. **No structural plasticity exercised.** All CartPole runs complete with the bootstrapped topology. Division, fusion, and clustering remain untested at scale under the new endo regime.
